# fork of [HTML5 Boilerplate](https://html5boilerplate.com)

Added:
- Bootstrap4
- Flexbox example
- gulp tasks for watch, sass...

### Getting started:
1. install dependencies

```
npm i
```

2.serve webapp

```
gulp serve
```



to read more about html5boilerplate goto https://github.com/h5bp/html5-boilerplate
